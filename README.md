# jmattheis/dotfiles

Managed with [twpayne/chezmoi](https://github.com/twpayne/chezmoi).

## Config

```ini
[data]
  email = "some@mail.tld"
  name = "Some Name"
  i3auto = ["nextcloud", "oni3start.sh"]
  pasink = "pulseaudiosink"
```

## Init

```bash
$ chezmoi init git@github.com:jmattheis/dotfiles.git
```
